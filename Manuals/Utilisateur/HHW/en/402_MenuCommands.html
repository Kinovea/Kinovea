Menu Commands



    This section reference Kinovea menus :
File, Edit, View, Image, Options, Help.


1. File

 OpenLaunches the "Open File" dialog box, allows you to choose a video and launch it into Kinovea.
 RecentLists the last successfully opened files.
 Close [file]Closes the specified video and its container screen.
 Save Video or Key Images Data...Launches the "Save" dialog box to save a movie from the current Working Zone.
 Load Key Images Data...Launches the Key Images Data files "Open" dialog, to import Drawings and comments.
 QuitCloses Kinovea.


2. Edit

 Undo [command]Reverts the last command.                                                                                                                            
 Redo [command]Redo the last canceled command.


3. View

 File ExplorerShows or hides the file explorer panel.                                                                                                     
 One Playback/Analysis ScreenOrganizes the workspace to display a Playback/Analysis screen.
If two screens are currently displayed, one will be automatically closed.
 Two Playback/Analysis ScreensOrganizes the workspace to display two Playback/Analysis screens.
 Invert Videos PositionsPuts the left video into the right screen and vice-versa.
 Show / Hide Common ControlsShows or hides the common controls panel, when two screens are displayed.


4. Image
   
 ColorsLaunches the dialog box that lets you tune images saturation.
 LuminosityLaunches the dialog box that lets you tune images luminosity.
 ContrastLaunches the dialog box that lets you tune images contrast.
 SharpnessLaunches the dialog box that lets you increase sharpness.
 MirrorFlips images horizontally
 Edges OnlyLaunches the dialog box that lets you create a black and white video where only edges are visible.
 GridDisplays a grid superimposed on images.
 3D PlaneDisplays a plane in 3D space superimposed on images.


5. Options

 LanguageChanges interface language.                                                                                                                           
 Check for available updatesLaunches the dialog box to check and download updates.
 Preferences...Launches the Preferences dialog to tweak settings.


6. Help
   
 Kinovea Help Launches user's manual                                                                                                                                 
 Video TutorialsLaunches the Help Videos dialog box.
 About KinoveaDisplays general informations about Kinovea.
