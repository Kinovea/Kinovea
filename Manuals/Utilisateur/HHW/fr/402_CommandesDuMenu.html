Commandes du Menu



    Cette description référence les menus de Kinovea :
Fichier, Édition, Affichage, Image, Options, Aide.


1. Fichier

 Ouvrir un Fichier Vidéo...Lance la boîte de dialogue d'ouverture de fichier, permet de choisir une vidéo et de la charger dans Kinovea.
 RécentsListe les derniers fichiers ouverts avec succès.
 Fermer [fichier]Ferme la vidéo spécifiée et l'écran qui le contient.
 Enregistrer la Vidéo ou
 les données des Images Clés...		Lance la boîte de dialogue d'enregistrement. 
  Charger des données d'Images Clés...Lance la boîte de dialogue d'ouverture de fichier de données d'Images Clés.
 QuitterFerme Kinovea.


2. Édition

 Annuler [commande]Annule la dernière commande effectuée.                                                                                                                        
 Refaire [commande]Réédite la dernière commande annulée.


3. Affichage

 Explorateur de FichierAffiche ou masque le panneau de l'explorateur de fichiers.                                                                       
 Un Écran de Lecture/AnalyseOrganise l'espace de travail pour afficher un écran de Lecture/Analyse.
Si deux écrans sont présents, l'un d'eux est automatiquement fermé.
 Deux Écrans de Lecture/AnalyseOrganise l'espace de travail pour afficher deux écrans de Lecture/Analyse.
 Inverser les Emplacement des VidéosPlace la vidéo de gauche dans l'écran de droite et vice-versa.
 Montrer/Cacher les Contrôles CommunsAffiche ou masque le panneau des contrôles communs, lorsque deux écrans sont visibles.              


4. Image
   
 Couleurs...Lance la boîte de dialogue permettant de modifier la saturation des images.
 Luminosité...Lance la boîte de dialogue permettant de modifier la luminosité des images.
 Contraste...Lance la boîte de dialogue permettant de modifier le contraste des images.
 Renforcement des Contours...Lance la boîte de dialogue permettant d'augmenter la visibilité des contours des images.
 MiroirInverse les images latéralement.
 Contours Seulement...Lance la boîte de dialogue permettant de créer une vidéo en blanc sur fond noir, où seuls les contours sont visibles.      
 GrilleAffiche une grille en surimpression sur l'image.
 Plan 3DAffiche un plan dans l'espace en surimpression sur l'image.


5. Options

 LanguesModifie la langue de l'interface.                                             
 Vérifier si des Mises à Jour
 sont Disponibles...		Lance la boîte de dialogue de vérification et téléchargement des mises à jour.  
 Préférences...Lance la boîte de dialogue de modification des préférences.


6. Aide
   
 Aide de Kinovea...Lance le manuel d'utilisation.                                                                                                                                   
 Tutoriels Vidéos...Lance la boîte de dialogue des vidéos d'aide intégrées.
 A Propos de Kinovea...Lance la boîte de dialogue d'informations sur Kinovea.
