A. General

    1. What is Kinovea ?

    Kinovea is a video analysis software dedicated to sport.

    2. Who is Kinovea for ?

    Kinovea targets primarily the coaches, athletes and medical professionals.

B. Installing / Updating

    3. Which platform are supported by Kinovea ?

    The operating systems supported by Kinovea are Windows XP and Windows Vista.
You need the framework .NET version 2.0 or higher to run Kinovea.

    4. What are the minimum requirements to run Kinovea ?

    The minimum requirements are the following :
CPU : 800 Mhz
RAM : 256 MB
Screen resolution : 1024x768 pixels
   
    5. How to install Kinovea ?

    An installer is available for download. Launch the installer and follow the instructions given.
If necessary, it is advised to install the .NET framework prior to proceeding to Kinovea install.

    6. How to update an old version of Kinovea ?

    You can use the update dialog box included into Kinovea to download the latest installer program.
Once downloaded, launch the installer and follow its instructions.

    8. How to contact Kinovea authors ?

    For software support, please leave a message on the forum at www.kinovea.org/en/forum.
For a more personal communication, send emails to infos@kinovea.org

    9. How can I contribute to the project ?

    You can do one of the following :
  • Report each and every malfunction encountered, bugs, lack of usability, texts mistakes.
  • Propose new functions or help other users on the community forum.
  • Translate the interface or the manual in your native language, or contact someone to do so.
  • Talk about the project to your coworkers and friends.
  • Modify the source code and send your improvements.
  • Donate hardware.
Please refer to the website, www.kinovea.org, for further informations.
