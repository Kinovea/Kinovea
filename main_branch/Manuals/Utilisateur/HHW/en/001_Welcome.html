    Thank you for installing Kinovea !

Kinovea is a video analysis software dedicated to sports motion.


How to use this manual ?

    The Frequently Asked Questions (FAQ) should answer the most common questions about Kinovea as a project.

If you are using Kinovea for the very first time, start with Visualizing Sport Motion.

In order to facilitate integration into your current work flow, a series of step by step tutorials are available.
They roughly match the main domains of sport analysis : Technical Observation and Analysis and Insights.

Configuration and maintenance tasks are gathered in General Tasks.

Lastly, the user interface is detailed in the Reference Guide, which you could consult on a per case basis.
