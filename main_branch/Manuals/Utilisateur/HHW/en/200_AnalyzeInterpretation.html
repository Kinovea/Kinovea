Analysis and Insights



    In addition to classic technical observation, biomechanics study give you a more formal analysis.
Analysis in Kinovea operates around the concept of Key Images. A Key Image can be set anywhere in the video.
It is a reference point to which you can attach a comment, draw free hand, measure an angle, etc.

Key Images data can be saved independently from the video or included into it to ease sharing.
In order to publish your analysis (or to finish it outside Kinovea), a set of saving functions are given.

Finally a Grid and a Plan in 3D space allow you to improve your observations.

Handling Key Images. (Add Key Images, Add Comments and Drawings, Save and Import Key Images data.)
Saving Still Images. (Single Image, Sequence of Images, Diaporama of Key Images, PDF document of Key Images.)
Saving Video Sequences. (Saving a Video, Formats and codecs used, Including Key Images data.)
Using the 2D Grid and the 3D Plane. (Showing the grid and the plan, Manipulating them, Modifying colors.)
