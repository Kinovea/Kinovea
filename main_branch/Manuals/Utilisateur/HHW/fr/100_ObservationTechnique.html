Observation Technique



    L'observation du geste sportif à l'aide du support vidéo est complémentaire de l'observation sur le terrain.
Ce chapitre regroupe quelques pages de pas-à-pas pour faire le lien entre ces deux modes d'étude.
 
Visualiser le Geste Sportif ( Ouvrir et lire une vidéo, sélectionner une zone de travail, ralentir le mouvement )
Décomposer la Technique ( Sélectionner une zone de travail, contrôler totalement la vitesse de déroulement de l'action )
Améliorer L'image ( Améliorer la saturation des couleurs, la luminosité, le contraste ou la définition )
Comparer ( Ouvrir deux vidéos, les synchroniser sur un évènement, afficher une vue miroir)
