Changer la langue de l'interface

    Pour changer la langue de l'interface, utilisez le menu : Options > Langues et choisissez parmi la liste.

Si vous avez choisi une langue dans laquelle vous ne savez plus utiliser l'interface, utilisez le raccourci clavier d'annulation CTRL+Z pour annuler le changement.
