A. Général
   
1. Qu'est ce que Kinovea ?

    Kinovea est un logiciel d'analyse vidéo spécialisé dans le geste sportif.       

2. A qui s'adresse Kinovea ?

    Kinovea s'adresse en premier lieu aux entraîneurs, aux sportifs et aux professionnels de santé.


B. Installation / Mise à jour

3. Quelles plateformes sont supportées par Kinovea ?

    Les systèmes d'exploitation supportés par Kinovea sont Windows XP et Windows Vista.
Vous devez disposer de la plateforme .NET version 2.0 ou supérieure pour faire fonctionner Kinovea.

4. Quelles sont les ressources systèmes minimales nécessaires à l'utilisation de Kinovea ?

    Les ressources minimales sont les suivantes :
Processeur : 800 Mhz
Mémoire Vive : 256 Mo
Résolution d'écran : 1024x768 pixels

5. Comment installer Kinovea ?

    Un programme d'installation est disponible au téléchargement. Lancez le programme d'installation et suivez les instructions
Si nécessaire, il est conseillé d'installer la plateforme .NET avant de procéder à l'installation de Kinovea.
 
6. Comment puis-je mettre à jour une ancienne version de Kinovea ?
   
    Vous pouvez utiliser la boîte de dialogue de mise à jour intégrée au logiciel pour télécharger le nouveau programme d'installation.
Une fois téléchargé, lancez le programme d'installation et suivez les instructions.

    
C. Divers
   
7. Peut-on utiliser Kinovea pour un usage commercial ou professionnel ?

    Oui.
L'utilisation de Kinovea est régie par la licence Gnu General Public Licence version 2.
La diffusion d'une version modifiée ou non de Kinovea nécessite la diffusion conjointe du code source.
Veuillez consulter l'article sur les Licences d'Utilisation pour prendre connaissance de l'intégralité des conditions.

8. Comment puis-je contacter les auteurs de Kinovea ?

    Pour le support logiciel lié à Kinovea veuillez laisser vos messages sur le forum à l'adresse www.kinovea.org/fr/forum.   
Pour une communication plus personnelle, envoyez un email à l'adresse suivante : infos@kinovea.org

9. Comment puis-je contribuer au projet ?
   
    Vous pouvez effectuer une ou plusieurs des actions suivantes :
  • Rapportez tout dysfonctionnement rencontré, bugs, manque d'ergonomie, textes approximatifs.
  • Proposez de nouvelles fonctions ou aidez d'autres utilisateurs sur le forum communautaire du projet.
  • Traduisez l'interface ou le manuel d'utilisation dans votre langue, ou contactez quelqu'un pour le faire.
  • Faîtes connaître le logiciel auprès de vos collègues et amis.
  • Modifiez le code source et proposez vos améliorations.
  • Donnez du matériel.
Consultez le site internet (www.kinovea.org) pour plus d'informations.
